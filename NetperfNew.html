# What's New With Netperf Checking this area from time to time will tell you what is new with the netperf benchmark. Announcements of new versions, and what has changed, as well as other interesting netperf news can be found here. ## The latest stable revision of netperf2 is 2.7.0 It is available at [ftp://ftp.netperf.org./netperf-2.7.0.tar.gz](ftp://ftp.netperf.org./netperf-2.7.0.tar.gz) If you prefer, you can grab the bits via [subversion](http://www.netperf.org/svn/netperf2/tags/netperf-2.7.0) ## July, 2015 Release 2.7.0 of netperf has shipped. Please see the <A HREF="http://www.netperf.org/svn/netperf2/tags/netperf-2.7.0/Release_Notes"> Release Notes </A> for more information. ## June, 2012 Release 2.6.0 of netperf has shipped. Please see the <A HREF="http://www.netperf.org/svn/netperf2/tags/netperf-2.6.0/Release_Notes"> Release Notes </A> for more information. ## July 19, 2011 Release 2.5.0 of netperf has shipped. Please see the <A HREF="http://www.netperf.org/svn/netperf2/tags/netperf-2.5.0/Release_Notes">Release Notes</A> for more information. ## June 11, 2009 Release 2.4.5 of netperf has shipped. Please see the <A HREF="http://www.netperf.org/svn/netperf2/tags/netperf-2.4.5/Release_Notes">Release Notes</A> for more information. ## February 15, 2007 Release 2.4.3 of netperf has shipped. Please see the <A HREF="http://www.netperf.org/svn/netperf2/tags/netperf-2.4.3/Release_Notes">Release Notes</A> for more information. ## October 18, 2005 Netperf.org has moved to a new system. This has enabled a multitude of changes: * A new <A HREF="ftp://ftp.netperf.org./">FTP download</A> location * A new <A HREF="http://www.netperf.org/svn/netperf2"> subversion</A> repository * A new mailing list <A HREF="http://www.netperf.org/pipermail/netperf-talk/">archive</A>. ## March 17, 2005 netperf is now using a configure script, has merged IPv4 and IPv6 tests and a host of other changes. See the release notes file for more. The netperf benchmark includes support for OpenVMS, and support for Linux sendfile(). ## February 14, 2003 Netperf now includes evolving support for OpenVMS and support for the Linux flavor of sendfile(). Additionally, fixes to set SO_REUSEADDR on the netserver listen socket, fixes for IPv6, and other sundries are included. The netperf benchmark includes the TCP_MAERTS test and IPv6 ## July 11, 2002 Netperf now includes a TCP_MAERTS test that has data flowing from the netserver to the netperf - this is opposite of the classic TCP_STREAM test. In theory, IPv6 support is integrated :) Also -DUSE_KSTAT for Solaris CPU util, and -DUSE_PROC_STAT for Linux. The netperf-talk mailing list now has an online archive. ## March 12, 1996 The netperf-talk mailing list now has an online <A HREF="./training/netperf-talk/index.html">archive</A> of discussions in netperf-talk. This archive will update in near real-time. More information can be found in the <A HREF="./training/netperf-talk/netperf-talk-archive.html">description</A> of the archive. ## February 1995 - Netperf 2.1 Released Thanks to the assistance of many folks on the net, revision 2.1 of netperf has released.
