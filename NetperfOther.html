# Other Network Performance Resources *This page has many outdated/broken links.* There are many other resources realted to network performance. It would be good for anyone wishing to measure network performance to be familiar with at least some of them. If you would like to find other Web sites mentioning/using netperf, any number of good search engines can be used to search for netperf. If you would like to find other benchmarks, you can follow the links from the <A HREF="./NetperfDownload.html">Download Netperf</A> page. Useful information about network performance testing can be found in [Benchmarking Terminology for Network Interconnection Devices (RFC 1242)](ftp://ds.internic.net/rfc/rfc1242.txt) That document is the first product of the [Benchmarking Methodology Working Group (BMWG)](http://www.ietf.cnri.reston.va.us/html.charters/bmwg-charter.html) of the <A HREF="http://www.ietf.cnri.reston.va.us/home.html"><IMG ALIGN="middle" SRC="http://www.ietf.cnri.reston.va.us/images/ietflogo.gif" ALT="Internet Engineering Task Force"></A>. <A HREF="./NetperfPage.html"> <IMG ALIGN="middle" SRC="./images/back.gif" alt="">Netperf Homepage</A>.
